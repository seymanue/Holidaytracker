# Holidaytracker
## Holidaytracker

Bu uygulama, Türkiye'nin resmi tatillerini public API üzerinden
çekerek konsol ekranında listeleyen bir C# uygulamasıdır.

### Kullanılan Teknolojiler
- C#
- .NET
- HttpClient
- JSON

### Kullanılan API
https://date.nager.at

### Uygulama Özellikleri
- Yıla göre tatil listeleme
- Tarihe göre tatil arama
- İsme göre tatil arama
- 2023–2025 yıllarına ait tüm tatilleri listeleme
